# pyisland-admin

`pyisland-admin` 是 Python-island 项目的后台管理与配套服务仓库，包含：

- `server/`：Spring Boot 后端服务（REST API，JWT 鉴权，阿里云 OSS 上传）
- `admin/`：后台管理前端（React + TypeScript + Vite）
- `docs/`：API 文档站（VuePress 2 + vuepress-theme-hope）

## 目录结构

```text
pyisland-admin/
  server/   # 后端服务（/api）
  admin/    # 管理后台前端
  docs/     # API 文档站
```
